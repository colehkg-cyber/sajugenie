# TASK: 타일러의 타이르는 운세 v3.0 전면 재설계

## 구조 변경
기존 6탭 → 3탭으로 변경

### Tab 1: 근본 (프로필 + 통합 운세)
입력 폼:
- 이름 (한자 한글발음 ex. 밝을현 준법준)
- 생년월일 (필수) 
- 태어난 시간 (선택)
- 얼굴 사진 (선택, Phase2)
- 손금 사진 (선택, Phase2)
- 타로 7장 고르기 (선택, Phase2)

입력 후 보여줄 것 (한 페이지에 통합):
- 명리학: 사주 분석 (천간/지지/오행/십신)
- 타로: 자동 3장 뽑기 (과거/현재/미래)
- 수비학: 생명수 계산
- 별자리: 12궁 해석
- 인상학: 오행별 얼굴형 매칭
- 최면심리학: 자기 암시 메시지

오늘의 운세 (프로필 입력 후):
- 내 띠가 맨 위에 나옴
- 아래에 12띠 전체 상세 운세 (공유용)
- 일진 기반, 띠별×연도별 해석
- 합충형파+오행관계 반영
- 형식: 날짜 + 일진 + 기운 해석 + 12띠 각각 (연도별 4개씩) + 운세지수

### Tab 2: 타이르는 타일러 도사
- 채팅 UI (Phase 1에서는 "준비 중" 표시)
- Phase 2에서 Claude API 연동

### Tab 3: 마이페이지
- localStorage에 프로필 저장
- 과거 운세 기록 보기
- 프로필 수정

## 디자인
- 기존 보라+골드 다크 테마 유지
- 하단 3탭 네비게이션
- 모바일 우선 반응형
- 별 배경 애니메이션 유지

## 기술
- Next.js + React + Tailwind
- localStorage로 데이터 저장 (API 없이)
- 정적 export (Vercel 배포)
